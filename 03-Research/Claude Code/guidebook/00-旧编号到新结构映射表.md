# Claude Code 源码导读手册旧编号到新结构映射表（初稿）

## 这份初稿怎么用

这是一版 **Phase 1 映射表初稿**，用途只有一个：

> 先把 `00-90` 这 91 篇正文，全部放进新手册结构里，明确它们各自属于哪一卷、在卷内排第几、是主链还是补充、后续要不要轻修。

这版先追求三件事：

1. 没有漏篇
2. 新卷结构可执行
3. 后续 guidebook 页面能直接接着做

这版还不是终稿，所以保留这些开放项：
- 个别标题简称后续可再统一
- 个别卷内顺序还可以微调
- `轻修` / `弱去重` 的边界后续可再收紧

---

## 输入集边界

本映射表的输入集是：
- `cc源码共读笔记/` 下的 **00-90 正文共 91 篇**

明确不计入正文输入集的文件：
- `README.md`
- `目录-00到90完整总表.md`
- 其他索引/目录类文件

---

## 字段说明

- **原编号**：旧系列编号
- **原标题简称**：为了映射表可读性做的短标题
- **新卷**：进入 guidebook 后所属卷
- **卷内顺序**：该卷里的阅读顺序
- **角色**：`主链` / `补充` / `卷末总结`
- **处理动作**：`保留` / `轻修` / `弱去重` / `补充阅读`
- **备注**：桥接、前置、后续会展开等说明
