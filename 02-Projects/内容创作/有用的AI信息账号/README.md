# 有用的AI信息账号

## 这个主题是什么

这是一个围绕“做一个对别人有用的 AI 信息账号”展开的长期内容主题。

它不以强人格、强日常、强陪伴感为主，而是把重点放在：

- AI 工作流
- Agent
- Memory / Knowledge Base / Skill
- 新机制与新接口的变化解读
- 真实可用的最佳实践

## 当前判断

这个账号不应该做成：

- 热点搬运号
- 纯教程说明书号
- 以个人状态输出为主的人格博主号

更适合做成：

> 一个持续提供判断、结构和最佳实践的 AI 工作方式情报转译账号。
