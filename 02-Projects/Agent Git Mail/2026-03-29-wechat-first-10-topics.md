# 公众号第一批 10 个选题

## 工作流重构

1. **AI 不是替你多做一点事，而是在重写你的工作流**
2. **哪些工作最适合先被 AI 重构，不是看重复度，而是看这 4 个信号**
3. **为什么很多人用了 AI 还是没变强：问题不在模型，在工作方式**
4. **一个工作流值不值得 agent 化，我现在主要看什么**

## 实战案例

5. **我最近怎么和 agent 一起做一个真实项目：从 brainstorm 到 coding 到 review**
6. **为什么我做了 Agent Git Mail：我不想把 agent 协作做成另一套 bureaucracy**
7. **我怎么判断一个 AI 项目该收口了，而不是继续发散**
8. **我为什么把 Hex 从“设计搭档”切回“执行搭档”**

## 角色升级与判断

9. **AI 时代，程序员最该升级的不是 coding，而是这 3 种能力**
10. **不要再用旧世界的胜负标准衡量自己**
