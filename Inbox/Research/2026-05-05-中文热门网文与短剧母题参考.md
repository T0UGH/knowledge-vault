## 最适合短剧拆解的三类母题

### 1. 都市逆袭 / 身份反转

参考：《赘婿》《庆余年》《大奉打更人》《夜的命名术》

短剧结构：

> 底层身份开局 → 被羞辱 → 暗藏真实能力 / 身份 → 小范围打脸 → 更大势力登场。

### 2. 女频重生 / 复仇 / 先婚后爱

参考：《坤宁》《折腰》《黑莲花攻略手册》《难哄》

短剧结构：

> 前世被骗 / 被害 → 重生醒来 → 第一集立刻改命 → 男主 / 反派关系错位 → 每集一个反转。

### 3. 悬疑超凡 / 副本闯关

参考：《诡秘之主》《灵境行者》《道诡异仙》《深海余烬》

短剧结构：

> 主角获得异常能力 → 进入规则世界 → 每集一个诡异任务 → 结尾揭开更大阴谋。
