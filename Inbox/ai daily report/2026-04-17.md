## GitHub 趋势项目
- **claude-mem** `claude-mem` 这周能进趋势榜，至少因为：`claude-mem` 是个围绕 `Claude Code` 的插件型项目。 另外，它会自动记录 Claude 在 coding session 里的操作。 同时，还会压缩记忆，并把相关上下文回注到后续 session。 [GitHub](https://github.com/thedotmack/claude-mem)
- **Archon** `Archon` 把自己定位成开源的 AI coding harness builder，主打让 AI coding 流程变得 deterministic、repeatable。 它上榜的理由也很具体：卖点不是再包一层 agent，而是把测试和执行流程做成可重复的基础设施。 [GitHub](https://github.com/coleam00/Archon)
- **multica** `multica` 这个 repo 把自己写成 open-source managed agents platform，目标是把 coding agents 变成能分派任务、跟踪进度、积累技能的真正队友。 [GitHub](https://github.com/multica-ai/multica)
- **andrej-karpathy-skills** `andrej-karpathy-skills` 提供的是单文件 `CLAUDE.md` 规则集，来源是 Andrej Karpathy 对 LLM coding pitfalls 的观察，目的就是直接改善 Claude Code 的行为。 [GitHub](https://github.com/forrestchang/andrej-karpathy-skills)
