Qwen

**Qwen OAuth has been removed.** The free-tier OAuth integration (`qwen-portal`) that used `portal.qwen.ai` endpoints is no longer available. See [Issue #49557](https://github.com/openclaw/openclaw/issues/49557) for background.

## [​](https://docs.openclaw.ai/providers/qwen#recommended-model-studio-alibaba-cloud-coding-plan)

Recommended: Model Studio (Alibaba Cloud Coding Plan)

Use [Model Studio](https://docs.openclaw.ai/providers/modelstudio) for officially supported access to Qwen models (Qwen 3.5 Plus, GLM-4.7, Kimi K2.5, MiniMax M2.5, and more).

```
# Global endpoint
openclaw onboard --auth-choice modelstudio-api-key

# China endpoint
openclaw onboard --auth-choice modelstudio-api-key-cn
```

See [Model Studio](https://docs.openclaw.ai/providers/modelstudio) for full setup details.
